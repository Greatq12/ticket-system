Helpdesk Ticketing System

Description
This is a simple web-based helpdesk ticketing system that simulates real IT support workflows.  
Users can create tickets, assign priorities, and track status through the ticket lifecycle.

This project was built to demonstrate practical helpdesk and IT support concepts.

---

Features
- Create support tickets
- Set priority (Low, Medium, High)
- Track ticket status (Open → In Progress → Resolved → Closed)
- Assign tickets to technicians
- Persistent storage using browser localStorage

---

Tech Stack
- HTML
- CSS
- JavaScript

---

How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
3. Start creating tickets
